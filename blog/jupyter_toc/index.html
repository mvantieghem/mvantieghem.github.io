How to make your jupyter notebook more like R markdown

Configurations and themes in jupyter notebooks

Make the transition to python easier for the R user

If you’re like me, and regularly used R markdown for analyses and reports, you were probably disappointed the first time you opened a jupyter notebook. After spending a few months struggling with python, I found that the aesthetics of my coding environment greatly improves my motivation to learn a new language. Fortunately, python users have developed themes and extensions to improve the jupyter notebook experience.

Step 1: Prettify your notebooks with themes

Step 2: Add table of contents

  1. Install package jupyter_contrib_nbextensions
pip install jupyter_contrib_nbextensions
jupyter contrib nbextension install --user
  1. Install the configurator (GUI to control extension options)
pip install jupyter_nbextensions_configurator
jupyter nbextensions_configurator enable --user
  1. Restart your notebook server & open Nbextensions
jupyter notebook

On “Home” jupyter page, click the (new) “Nbextensions” tab This opens a new page with all possible extensions available.

  1. Configure your Table of contents

Click the check mark on ‘table of contents’ box, and scroll down to view the parameter options Select check marks for “Add table of contents cell at top of notebook”, and any other TOC features you want! (I also like the floating TOC sidebar).
